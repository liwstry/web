# Интернет магазин "АвтоМир"
-  Веб-приложение для заказа автомобилей

## Технологии

#### Python 3.12.6
- Веб-фреймворк: `Flask 3.1.3`
    - _ORM:_ `Flask-SQLAlchemy 3.1.1` + `SQLAlchemy 2.0.48`
    - _Работа с пользовательской сессией:_ `Flask-Login 0.6.3`
    - _Отправка сообщений на почту:_ `Flask-Mail 0.10.0`
    - _Поддержка RealTime:_ `Flask-SocketIO 5.6.1`
- Работа с временными токенами: `itsdangerous 2.2.0`
- Генерация секретного ключа: `secrets`
- Работа с переменными .env: `python-dotenv 1.2.2`
- Веб запросы: `requests 2.33.0`
- Получение данных состояний компонентов ПК: `psutil 7.2.2`
- Хэширование: `werkzeug 3.1.7`
- Валидация:
    - _Регулярные выражения:_ `re`
    - _Валидация почты:_ `email-validator 2.3.0`
- Перевод текста на RU: `eng_to_ru 0.1.1`

## Запуск проекта

1. Установить зависимости из **requirements.txt**: `pip install -r requirements.txt`
2. Запустите приложение введя в терминале: `python app.py`
3. Приложение будет доступно по адресу: `http://127.0.0.1:5000`

### Дополнительно

1. Для отправки сообщений на почту:
    - Если в репозитории нет файла **.env**, то создайте его в корне проекта со следующей структурой:
        ```
        EMAIL_ADDRESS = "Почта (gmail)"
        MAIL_PASSWORD = "Пароль приложений (smtp)"
        API_TOKEN_WEATHER = "API ключ (openweathermap)"
        ```
    - В файле **.env** введите свою почту от gmail в переменную `EMAIL_ADDRESS`
    - Получите пароль приложений (**google smtp**) и введите его в переменную `MAIL_PASSWORD`

2. Для работы с **API** погоды:
    - Получите **API** ключ на [openweathermap](https://openweathermap.org/)
    - Введите его в файле **.env** в переменную `API_TOKEN_WEATHER`
