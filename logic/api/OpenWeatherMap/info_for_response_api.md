# Информация по JSON‑ответу от OpenWeatherMap

## 1. coord — координаты города

* `lon` (longitude) — долгота: $37{,}6156^\circ$ в. д.
* `lat` (latitude) — широта: $55{,}7522^\circ$ с. ш.

## 2. weather — массив с общей информацией о погоде

Может содержать несколько элементов при смешанных условиях.

* `id` — внутренний код типа погоды (804 = сплошная облачность).
* `main` — основная категория: `"Clouds"` (облачность).
* `description` — детальное описание: `"overcast clouds"` (сплошная облачность).
* `icon` — код иконки для отображения: `"04d"` (иконка сплошной облачности днём).

## 3. base — источник данных

* Указывает, что данные получены от метеорологических станций.

## 4. main — основные метеорологические параметры

* `temp` — текущая температура в кельвинах: $286{,}39\ \text{K}$ ($\approx 13{,}24^\circ \text{C}$).
* `feels_like` — «ощущается как» в кельвинах: $285{,}13\ \text{K}$ ($\approx 11{,}98^\circ \text{C}$).
* `temp_min` — минимальная температура за сутки в кельвинах.
* `temp_max` — максимальная температура за сутки в кельвинах.
* `pressure` — атмосферное давление на уровне моря в гПа (гектопаскалях) или мбар.
* `humidity` — влажность воздуха: 52 %.
* `sea_level` — давление, приведённое к уровню моря (гПа).
* `grnd_level` — давление на уровне земли (гПа).

## 5. visibility — видимость

* Видимость в метрах

## 6. wind — параметры ветра

* `speed` — скорость ветра в м/с: $3{,}19\ \text{м/с}$.
* `deg` — направление ветра в градусах: $31^\circ$ (северо‑северо‑восток).
* `gust` — порывы ветра в м/с: $5{,}03\ \text{м/с}$.

## 7. clouds — облачность

* `all` — процент покрытия неба облаками: 100 % (полностью затянуто).

## 8. dt — временная метка

* Время измерения в формате Unix‑timestamp (секунды с 01.01.1970).
* Чтобы преобразовать в читаемый формат:
    * используйте онлайн‑конвертеры;
    * в Python: `from datetime import datetime; datetime.fromtimestamp(1774688795)`.

## 9. sys — системные данные

* `type` — тип внутренней системы (служебное поле).
* `id` — идентификатор системы.
* `country` — страна: `"RU"` (Россия).
* `sunrise` — время восхода солнца (Unix‑timestamp).
* `sunset` — время заката солнца (Unix‑timestamp).

## 10. Прочие поля

* `timezone` — смещение в секундах от UTC: 10 800 с (= 3 часа, UTC+3).
* `id` — уникальный ID города в базе OpenWeatherMap: 524 901 (Москва).
* `name` — название города: `"Moscow"`.
* `cod` — HTTP‑статус ответа API: 200 (успех).
